# xcp-lite - all_features_demo

Comprehensive reference application that exercises the full xcp-lite feature surface in a single
program: basic types, nested structs, arrays, matrices, arrays of structs, curves/maps with shared
axes and multi-threaded measurement.

This is the demo paired with the CANape project in `./CANape`.

Run:

```
cargo run -p all_features_demo
```

See [the examples overview](../README.md) for common build, run and command line options
(including `--flatten`).



## CANape

To use one of the CANape projects included, use 'Project/Open" and select the file CANape.ini in the CANape folder.  

The examples are build with CANape 23.
Older versions were not tested.
